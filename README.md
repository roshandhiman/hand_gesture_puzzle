Hand Gesture Puzzle Game

Hand Gesture Puzzle Game is a Python project that uses computer vision to create an interactive puzzle controlled by hand gestures. The application captures an image from the webcam and splits it into a 3x3 puzzle grid. The puzzle pieces are shuffled and the user must rearrange them to form the original image.

The system uses MediaPipe for hand tracking and OpenCV for image processing. The user can control the puzzle by making hand gestures in front of the camera. A timer of two minutes is given to solve the puzzle.

Technologies Used
Python
OpenCV
MediaPipe
NumPy

How to Run
1. Install Python
2. Install required libraries: opencv-python, mediapipe, numpy
3. Run the main file using:
python puzzle_game.py

Features
Live webcam capture
3x3 puzzle generation
Hand gesture interaction
2-minute timer
Puzzle solved detection

This project demonstrates real-time computer vision, gesture recognition, and interactive game development using Python.
